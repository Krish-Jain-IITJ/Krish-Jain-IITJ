# Krish Jain
**B.Tech Electrical Engineering · IIT Jodhpur**

Researcher at the intersection of **control theory**, **reinforcement learning**, and **autonomous systems**. I build systems that are safe, mathematically grounded, and deployable in the real world — from power grids and lunar rovers to unicycle robots navigating cluttered environments.

---

## Research Interests

- Hybrid control architectures (classical + learned)
- Safe reinforcement learning & Barrier Lyapunov Functions
- Autonomous navigation and trajectory tracking
- Remote sensing & planetary robotics
- AI-powered simulation and digital twins

---

## Selected Projects

| Project | Description | Stack |
|---|---|---|
| [Residual RL for Obstacle Avoidance](https://github.com/Krish-Jain-IITJ/Residual_RL) | BLF + SAC hybrid controller for unicycle robots; 99.96% collision-avoidance rate | Python, PyTorch |
| [Power Grid DQN](https://github.com/Krish-Jain-IITJ/Power_Grid) | DQN agent managing a 50-substation grid over 8,760 hours of simulation | C++, Python, ZMQ |
| [Mission Chandrendra](https://github.com/Krish-Jain-IITJ) | End-to-end lunar ice detection & autonomous landing pipeline | Python, QGIS, Gazebo |
| [Circuit Theory Platform](https://github.com/Krish-Jain-IITJ/Circuit-Theory) | AI-powered web platform for circuit theory learning with RAG & CV | Django, LangGraph |
